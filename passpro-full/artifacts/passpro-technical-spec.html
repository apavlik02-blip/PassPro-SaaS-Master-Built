01 — Product Overview

What PassPro is built to do

PassPro is a modern, AI-powered insurance licensing exam preparation platform. Unlike legacy competitors (ExamFX, Kaplan), PassPro is built mobile-first, gamified, and powered by two breakthrough AI systems: an adaptive study companion and a customer roleplay simulator.

💡 Core thesis: Every other platform prepares you to pass the exam. PassPro prepares you to pass the exam AND succeed in the job. This is the entire market positioning.

The 4 Core Problems We Solve

😴

Boring Content

Static walls of text replaced with bite-sized micro-lessons, real-world scenarios, and short explainer videos.

📊

No Progress Visibility

Real-time readiness scores, topic-level mastery tracking, and predicted pass probability updated after every session.

🎮

Zero Engagement

Daily streaks, XP points, badges, leaderboards, and study challenges keep learners returning every day.

🎯

No Smart Remediation

After every practice exam, instant AI gap analysis identifies weak areas and auto-builds a personalized catch-up plan.

06 — AI Systems Specification

How the AI is built

AI Study Companion — System Prompt Architecture

The companion is powered by Claude API with a carefully engineered system prompt. The prompt contains three layers:

SYSTEM AI Companion Prompt Structure
Layer 1 — Identity:
  "You are Coach, an expert insurance licensing tutor..."
  "Tone: encouraging, clear, never condescending..."

Layer 2 — Knowledge Base:
  "[Injected: full state curriculum for {state}]"
  "[Injected: student's weak areas from DB]"
  "[Injected: last 5 session results]"

Layer 3 — Behavioral Rules:
  "Never give answers directly — use Socratic method"
  "Always end with a follow-up question or drill offer"
  "If student is frustrated, acknowledge before explaining"

AI Roleplay Simulator — Persona System

Each customer persona is a separate, fine-tuned system prompt with defined personality, objections, emotional state, and knowledge level. The AI stays in character throughout the session.

08 — API Reference

Core API endpoints

RESTful API built with Node.js + Express. All endpoints require Bearer token authentication except public routes.

Authentication

POST/api/auth/registerCreate new user account
body: { email: string, password: string, target_state: string, plan_id: string }
POST/api/auth/loginLogin + return JWT

Study & Exams

GET/api/modules/:stateGet all modules for a state
GET/api/exam/generateGenerate adaptive practice exam
query: { state: string, focus_weak: boolean, question_count: number }
POST/api/exam/submitSubmit exam + trigger gap analysis
body: { exam_id: string, answers: object[], time_taken_sec: number }

AI Services

POST/api/ai/companion/messageSend message to AI study companion
body: { message: string, session_id: string, context: object }
POST/api/ai/roleplay/messageSend message in roleplay session
body: { message: string, session_id: string, persona_id: string }
POST/api/ai/roleplay/scoreScore completed roleplay session
body: { session_id: string } → returns full scoring breakdown + coaching notes

Progress & Analytics

GET/api/progress/dashboardGet full dashboard data for user
GET/api/agency/:id/teamGet all team member progress (admin only)
09 — Tech Stack

Every technology and why

Frontend
Next.js 14
React framework with server-side rendering. Fast, SEO-friendly, and Vercel-native for easy deployment.
Styling
Tailwind CSS
Utility-first CSS. Fast to build, easy to maintain, consistent design system out of the box.
Backend
Node.js + Express
Proven, fast API layer. Huge ecosystem. Easy to find developers. TypeScript for type safety.
Database
Supabase (Postgres)
Managed PostgreSQL with built-in auth, real-time, and row-level security. Saves months of DB infrastructure work.
AI Engine
Claude API
Best-in-class for nuanced, educational conversations. Long context window ideal for maintaining session history.
Auth
Clerk
Drop-in auth with social login, MFA, and org management for agency accounts. Saves 2–3 weeks of build time.
Payments
Stripe
Industry standard. Handles one-time purchases, subscriptions, and agency billing without custom code.
Hosting
Vercel + Railway
Vercel for frontend (zero-config), Railway for API. Both auto-scale and have generous free tiers to start.
Email
Resend
Modern email API with React email templates. Handles transactional and marketing emails cleanly.
